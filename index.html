My Journey to CS

Graduating Longwood University with a bachelor's of science in chemistry and biology, I went to pharmacy school at Northeastern University and entered as the top 10-15% of my class on scholarship. In 2020, I changed my career to a path in CS. I am interested in medtech and applying knowledge and experience within science/healthcare to areas in CS.

Featured Projects

Recognition using Deep Networks

I developed a project to build, train, analyze and modify a deep network for recognition task. This project used MNIST digit recognition data set and implemented with Python and PyTorch.

Calibration and Augmented Reality

I created an app to calibrate a camera, and use the calibration to generate virtual objects in a scene. I used C++ and openCV library.

Mobile App: EyesUp

I indpedently organized and developed group mobile app to act as a simple reminder for users to look up (& prevent eye strain) with mobile devices. EyesUp was created using Java and ReactNative.

Physical Activity Tracker App

I created an app that tracks activity data, and is implemented using Java, Spring Framework, and utilizes mySQL database. Graph visualizations were created using Chart.js.
